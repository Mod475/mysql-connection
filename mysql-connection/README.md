# مشروع اتصال MySQLi

## وصف المشروع
مشروع عملي يوضح كيفية الاتصال بقاعدة البيانات باستخدام MySQLi في PHP، مع تطبيق نظام CRUD كامل.

## المميزات
- اتصال آمن مع MySQLi
- حماية من هجمات SQL Injection
- دعم اللغة العربية
- تصميم متجاوب
- فصل الملفات (نمط MVC مبسط)
- معالجة الأخطاء
